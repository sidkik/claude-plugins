# sidkik-plugins

Claude Code plugin marketplace for Sidkik.

```bash
claude plugin marketplace add sidkik/claude-plugins
```

## Plugins

| Plugin | Description |
|---|---|
| [codex-crew](codex-crew/) | Purpose-built Codex delegate agents (implementer / reviewer / fast) with pinned models, invocable like native subagents, riding the official `codex@openai-codex` plugin's companion runtime |

```bash
claude plugin install codex-crew@sidkik-plugins
```
