# sidkik-plugins

Claude Code plugin marketplace for Sidkik.

```bash
claude plugin marketplace add sidkik/claude-plugins
```

## Plugins

| Plugin | Description |
|---|---|
| [codex-crew](codex-crew/) | Tiered Codex delegate agents (implementer-sol / implementer-terra / implementer-luna / reviewer) across the GPT-5.6 ladder at xhigh effort, with per-tier selection criteria in each agent description, riding the official `codex@openai-codex` plugin's companion runtime |

```bash
claude plugin install codex-crew@sidkik-plugins
```
